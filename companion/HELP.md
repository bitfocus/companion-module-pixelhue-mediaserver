## PixelHue

This module will allow you to control the following PixelHue media server products: MS3.

### Configuration

- Enter the IP address of the product in the configuration settings.

**Available actions:**

- Program
- Pause_program
- Play_program
- Stop_program
- Open_ftb
- Close_ftb
- Open_volume
- Close_volume
- ppt_pgup
- ppt_pgdn
- Volume_up
- Volume_down
